# line-notify
LineNotify を利用するための PHP ライブラリ

## Support versions
* PHP8 <= 8.1

## Install

利用するプロジェクトの `composer.json` に以下を追加する。
```composer.json
"repositories": {
    "line-notify": {
        "type": "vcs",
        "url": "https://github.com/shimoning/line-notify.git"
    }
},
```

その後以下でインストールする。

```bash
composer require shimoning/line-notify
```
